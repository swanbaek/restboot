# restboot
- Spring Boot
- Swagger
- Security
## 핵심 구현 내용
-  회원가입 처리 (비밀번호 암호화 포함)
-  로그인시 role-USER인 경우 로그인 처리
-  사용자별 접근 권한 제어 (일반 USER와 관리자 ADMIN으로 나눠 접근 제어 처리)
